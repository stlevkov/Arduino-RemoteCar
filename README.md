# Arduino-RemoteCar-433MHz
Simple code for using Servo, Motor, Arduino and 433MHz transmitter

<h2> Hardware used: </h2>
<ol>
  <li>Arduino Leonardo</li>
  <li>24l01 Tranciever</li>
  <li>Mini Servo</li>
</ol>
